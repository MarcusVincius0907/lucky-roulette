
# Canoe Reddit Posts



## Tech Stack

**Client:** Vue, Typescript, Vuex, Tailwindcss, Vuetify

**Server:** Node, Express


## Run Locally

Clone the project

```bash
  git clone https://github.com/MarcusVincius0907/canoe-reddit-posts.git
```

Go to the project directory

```bash
  cd canoe-reddit-posts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
